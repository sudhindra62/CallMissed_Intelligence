# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added
- **Core Features:** Robust foundational setup using Vite, React, and Express.
- **Architecture:** Full-stack design ensuring API keys are securely managed server-side.
- **Streaming:** High-performance, Server-Sent Events (SSE) based text streaming for chat.
- **Vision:** Multi-modal support allowing users to upload or paste images for AI analysis.
- **Image Generation:** Text-to-image capabilities utilizing the `flux-2-klein-9b` model.
- **Documentation:** Comprehensive `README.md`, `SECURITY.md`, and `ARCHITECTURE.md`.
- **Security:** In-memory rate limiting and Zod-based payload validation.
- **Performance Optimization:** Lazy loading for heavy Markdown renderers and dialogs.
- **Accessibility:** Full keyboard navigation, ARIA tags, and screen-reader compatibility.
